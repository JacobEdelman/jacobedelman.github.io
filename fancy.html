I am president and founder of HSCS.io, an organization to promote high school computer science. HSCS.io works with high school students to bring multiple competitions to high schools across the world. I created it after running HSCTF, a high school programming competition that had over 2000+ students participate and was sponsored by Facebook and others. With HSCS.io I’ve also helped run EasyCTF, CTCTF, and sCTF.

You can see all my projects on my github account

Seeding.rocks

At PennApps Spring 2014 Spring my team and I built Seeding.rocks, a technology that allows website distribution in a peer to peer format, in any web browser. This means that rather than having a server do the grunt work of distributing a website the work is democratized. This greatly reduces server load.

WordyScript

WordyScript is a super-set of python that my team and I created along with a website for HackBCA 2014, winning the Namecheap Prize and being in the top 20. The language makes it easier to code in python with several syntactical changes making WordyScript an effective educational tool. Users of the site can execute code, convert it to JavaScript, or share it with friends.  

Lark

Lark is an experimental programming language I created and have been working on. It is based on mutable syntax and is aimed towards uses involving fast prototyping and experimentation with programming language features. It’s still very early in development and a v0.1.0 is currently in the works.
